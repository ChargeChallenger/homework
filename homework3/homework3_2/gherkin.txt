Сценарий: Выбран верный пункт меню
        Дано программа выводит меню и просит выбрать пункт меню
        Когда пользователь нажимает key
        То выводит result

	Примеры:
            | key	    | result	                         |
            | 1             | Вывод списка форматов файла        |
            | 2             | Вывод формата по краткому описанию |
            | 3             | Добавление нового формата          |
            | Любая клавиша | Возврат в главное меню             |

Сценарий: Поиск формата файла
        Дано просят ввести краткое описание формата
        Когда пользователь вводит shortDescription
        То выводит shortDescription: fullDescription

	Примеры:
            | shortDescription | fullDescription |
            | .docx            |     document    |
            | .exe             |   executable    |

Сценарий: Выбор несуществующего пункта меню
	Дано выводится меню и просят выбрать пункт меню
        Когда нажимаем key
        То выводит result

	Примеры:
            | key | result |
            | r   |	   |
            
Сценарий: Формат файла не найден
        Дано просят ввести краткое описание
        Когда вводим shortDescription
        То выводит result

	Примеры:
            | shortDescription | result      		   |
            | .jpg    	       | Введен отсутствующий ключ |
            | .exexe           | Введен отсутствующий ключ |
